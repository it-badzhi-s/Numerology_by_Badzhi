# ✅ Чек-лист запуска Нумерологического Бота

## Перед запуском

### 📋 Подготовка
- [ ] Установлен Python 3.7 или выше
- [ ] Есть аккаунт в Telegram
- [ ] Есть доступ к интернету

### 🤖 Создание бота
- [ ] Открыт чат с @BotFather
- [ ] Отправлена команда `/newbot`
- [ ] Придумано имя бота
- [ ] Выбран username (заканчивается на `bot`)
- [ ] Скопирован токен в безопасное место
- [ ] *(Опционально)* Установлено описание бота
- [ ] *(Опционально)* Загружено изображение профиля

### 💻 Настройка кода
- [ ] Скачан/скопирован весь проект `numerology_bot/`
- [ ] Открыт файл `config.py`
- [ ] Вставлен токен вместо `YOUR_BOT_TOKEN_HERE`


## Решение проблем

### ❌ Бот не запускается

**Ошибка: "ModuleNotFoundError: No module named 'telegram'"**
```bash
# Решение:
pip3 install python-telegram-bot --break-system-packages
```

**Ошибка: "telegram.error.InvalidToken"**
```
# Решение:
1. Проверьте токен в config.py
2. Убедитесь, что скопировали весь токен целиком
3. Проверьте отсутствие лишних пробелов
```

**Ошибка: "BOT_TOKEN = YOUR_BOT_TOKEN_HERE"**
```
# Решение:
Вы забыли заменить токен в config.py!
```

### ⚠️ Бот не отвечает

**Бот найден, но не отвечает на /start**
```
# Проверьте:
1. Запущен ли скрипт bot.py?
2. Нет ли ошибок в терминале?
3. Есть ли интернет-соединение?
```

**Бот перестал отвечать после изменений в коде**
```
# Решение:
1. Остановите бота (Ctrl+C)
2. Проверьте синтаксис Python
3. Запустите заново: python3 bot.py
```

### 🐛 Некорректные результаты

**Неправильные расчёты**
```
# Проверьте:
1. Функцию calculate_numerology_sum()
2. Правильность извлечения цифр
3. Логику приведения к мастер-числам
```

**Отсутствует текст толкования**
```
# Проверьте:
1. Файлы в папке data/
2. Словари PHONE_MEANINGS, ADDRESS_MEANINGS, BIRTH_MEANINGS
3. Наличие всех чисел (1-9, 11, 22, 33)
```
