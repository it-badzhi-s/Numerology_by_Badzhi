# 🔮 Нумерологический Telegram Бот - Обзор проекта

## 📊 Статистика проекта

- **Язык:** Python 3.7+
- **Библиотека:** python-telegram-bot v20+
- **Файлов кода:** 8 Python файлов
- **Файлов документации:** 6 MD файлов
- **Строк кода:** ~800+ строк
- **Типов расчётов:** 3 (телефон, адрес, дата рождения)
- **Нумерологических значений:** 36 (12 значений × 3 типа)
- **Уровень сложности:** Начальный → Средний

## 🎯 Цели проекта

### Основная цель
Создать простого и интуитивного Telegram бота для нумерологических расчётов с возможностью лёгкого расширения функционала.

## 💡 Основные компоненты

### 1. Меню и навигация (menu.py)
- Главное меню с кнопками
- Обработка команд /start, /menu
- Информация о боте
- Возврат в меню

### 2. Расчёты (calculations.py)
- 3 типа расчётов
- ConversationHandler для диалогов
- Валидация ввода
- Нумерологический алгоритм

### 3. База знаний (data/)
- 12 значений на каждый тип
- Подробные описания
- Практические рекомендации
- Мастер-числа (11, 22, 33)

## 🧮 Нумерологический движок

### Алгоритм расчёта
```python
Извлечение цифр → Суммирование → Приведение к 1-9
                                   ↓
                          Сохранение 11, 22, 33
```

### Поддерживаемые форматы
- **Телефон:** +7(999)123-45-67, 89991234567
- **Адрес:** ул. Ленина д.25 кв.17
- **Дата:** 15.03.1990, 15/03/1990
