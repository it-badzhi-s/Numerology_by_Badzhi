# 🔮 Телеграм Бот для Нумерологических Расчётов

Бот для выполнения нумерологических расчётов на основе номера телефона, адреса и даты рождения.

## 📁 Структура проекта

```
numerology_bot/
├── bot.py                     # Главный файл бота
├── config.py                  # Конфигурация и настройки
├── handlers/                  # Обработчики
│   ├── menu.py               # Меню и навигация
│   └── calculations.py       # Логика расчётов
└── data/                      # Нумерологические данные
    ├── phone_meanings.py     # Значения для телефонов
    ├── address_meanings.py   # Значения для адресов
    └── birth_meanings.py     # Значения для дат рождения
```

## 🚀 Быстрый старт

### 1. Создание бота в Telegram

1. Откройте Telegram и найдите **@BotFather**
2. Отправьте команду `/newbot`
3. Придумайте имя для бота (например, "Numerology Helper")
4. Придумайте username для бота (должен заканчиваться на `bot`, например, `mynumerology_bot`)
5. **Скопируйте токен**, который выдаст BotFather

### 2. Установка зависимостей

```bash
pip3 install python-telegram-bot --break-system-packages
```

### 3. Настройка токена

Откройте файл `config.py` и замените `YOUR_BOT_TOKEN_HERE` на ваш токен:

```python
BOT_TOKEN = "1234567890:ABCdefGHIjklMNOpqrsTUVwxyz"  # Ваш токен
```

### 4. Запуск бота

```bash
cd numerology_bot
python3 bot.py
```

Вы увидите сообщение:
```
🚀 Бот запущен и готов к работе!
```

## 📱 Использование бота

### Команды:
- `/start` - Запуск бота и приветствие
- `/menu` - Показать главное меню
- `/cancel` - Отменить текущую операцию

### Доступные расчёты:

1. **📱 По номеру телефона**
   - Введите номер в любом формате
   - Получите нумерологическое значение

2. **🏠 По адресу**
   - Введите ваш адрес
   - Узнайте энергию вашего дома

3. **🎂 По дате рождения**
   - Введите дату в формате ДД.ММ.ГГГГ
   - Откройте ваше Число Жизненного Пути

## 🔧 Настройка и расширение

### Добавление собственных толкований

Все тексты нумерологических значений находятся в папке `data/`:

- `phone_meanings.py` - толкования для номеров телефонов
- `address_meanings.py` - толкования для адресов
- `birth_meanings.py` - толкования для дат рождения

Просто отредактируйте нужный файл и добавьте свои тексты.

### Изменение приветственного сообщения

Откройте `config.py` и измените переменные:
- `WELCOME_MESSAGE` - приветствие при /start
- `MENU_TEXT` - текст главного меню

## 📊 Как работает нумерология в боте

Бот использует классический метод приведения чисел:
1. Извлекает все цифры из введённых данных
2. Суммирует их
3. Приводит к однозначному числу (1-9) или мастер-числу (11, 22, 33)

**Пример:**
- Телефон: +7 999 123 45 67
- Цифры: 7999123456 7
- Сумма: 7+9+9+9+1+2+3+4+5+6+7 = 62
- Приведение: 6+2 = **8**

## 🎨 Возможности расширения

Вы можете легко добавить новые типы расчётов:

1. Создайте новый файл в `data/` с толкованиями
2. Добавьте обработчик в `handlers/calculations.py`
3. Добавьте кнопку в меню в `handlers/menu.py`
4. Зарегистрируйте обработчик в `bot.py`

- Квадрат Пифагора

## 🐛 Решение проблем

### Бот не запускается
- Проверьте, что токен правильно указан в `config.py`
- Убедитесь, что установлена библиотека `python-telegram-bot`

### Бот не отвечает
- Проверьте интернет-соединение
- Убедитесь, что бот запущен в терминале
- Проверьте логи в терминале

### Ошибка при установке библиотеки
На macOS M1 используйте флаг `--break-system-packages`:
```bash
pip3 install python-telegram-bot --break-system-packages
```
